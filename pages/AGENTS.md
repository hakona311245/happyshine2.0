# AGENTS.md

## Project

This is a React + Vite + TypeScript project.

## Commands

- Build: `npm run build`
- Dev: `npm run dev`
- Type check: `npx tsc --noEmit`

## Rules for Codex

- Do not edit unrelated files.
- Do not rewrite large sections unless I ask.
- Do not change Vietnamese text unless I ask.
- Do not change routes unless I ask.
- Do not add new dependencies unless necessary.
- Keep TypeScript safe.
- Preserve existing design style unless I ask for a redesign.

## UI Rules

When improving UI:

- Improve spacing, layout, visual hierarchy, and responsiveness.
- Make the page work well on mobile and desktop.
- Keep Vietnamese text readable.
- Avoid fonts that break Vietnamese accents.
- Use the existing color palette first.
- Add hover/focus states for buttons and links.
- Keep animations subtle.

## Before big changes

For large UI changes or changes across multiple files:

1. Inspect the relevant files first.
2. Give me a plan before editing.
3. Tell me which files will be changed.
4. Wait for my approval before editing.

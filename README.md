
# Happy Shine English Center

A bright, modern website for the Happy Shine English Center. Built with React, TypeScript, and TailwindCSS to showcase courses, student achievements, certifications, and contact information in a clean, responsive layout.

## Quick Start

Prerequisites: Node.js

1. Install dependencies: `npm install`
2. Run the app: `npm run dev`
3. Open the local URL shown in your terminal.

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- React Router

## Build

`npm run build`
